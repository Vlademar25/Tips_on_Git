\### Базовая настройка Git и инициализация локального репозитория на Windows.



\#### Приветствую! В этой краткой инструкции опишем шаги для настройки Git с нуля на Windows и инициализации локального репозитория.



\##### Загрузка и установка Git Bash.

1. Перейти по ссылке \[Git](https://git-scm.com/install/windows).
2. Выбрать подходящий дистрибутив и загрузить.
3. Запустить установщик и Проверить путь установки (изменить, если дефолтный вариант не подходит).
4. На вкладке с выбором компонентов "Select Components" Проверить, что включена опция "Git Bash Here" (эта опция позволит запускать консоль из любой папки).
5. Остальные опции рекомендуется оставить по умолчанию.



Мы успешно установили Git Bash!



\##### Инициализация локального репозитория.

1. Запустите Git Bash (Поиском или Запустив из папки Git в списке всех программ меню Пуск).
2. Узнаем в какой директории по умолчанию открылся Bash. 
3. Используем команду "pwd" (от англ. present working directory).
4. Теперь нужно перейти в директорию, где будет инициализирован репозиторий. 
Выберите свою рабочую папку.
5. Переходим в папку с помощью команды "cd <Путь к папке>" (от англ. change directory).
К примеру, "cd /c/Users/<Current User>/<Project\_Folder>"
6. Создаём папку для репозитория с помощью команды "mkdir" (от англ. make directory)
К примеру, "mkdir First\_Project".
7. Переходим в созданную директорию с помощью команды "cd <Имя созданной папки>".
8. Инициализируем репозиторий с помощью команды "git init".

>> Репозиторий инициализирован. Должно отобразится сообщение с успешной инициализацией пустого репозитория.
